# vsCode-config
Memo configuration pour VSCode

## Association extension/langage
Pour associer un langage avec une extension, placer ce code dans le fichier config de VSCode.

    {
        "files.associations": {
        "*.extension": "langage",
        }
    }
    
### Correspondances
|Extension|   Langage   |
|---------|-------------|
|  *.html |    html     |
|  *.html | django-html |
|  *.css  |     css     |
|  *.py   |    python   |
|  *.php  |     php     |
|  *.js   | javascript  |
|  *.json |    json     |
